# Bing Daily Wallpaper App

A Windows system tray application that automatically sets your desktop background to the daily Bing image.

## Features
- **Daily Updates**: Automatically checks for and downloads the latest Bing wallpaper every 15 minutes.
- **System Tray Icon**: Sits quietly in your taskbar (near the clock).
- **Preview Gallery**: View the current image and a history of previous images via the tray menu.
- **Startup**: Automatically launches on login (configurable).
- **No Console**: Runs silently in the background without annoying popup windows.

## Requirements
- Windows 10 or 11
- Python 3.x (if running from source)
- Dependencies: `pystray`, `Pillow`, `requests`
